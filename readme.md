# 可愛鬆餅機器人

## 請注意！
inline 功能由於 API 限制需要於 `config.js` 填寫分流用聊天室（開幾個頻道或群組給機器人發垃圾啦）

若不須使用該功能可直接於 BotFather 關閉

## Usage

```sh
$ npm install
$ BOT_TOKEN='123:......' npm run dev
```

```sh
$ yarn
$ BOT_TOKEN='123:......' yarn dev
```
